# Usage
<!-- usage -->
```sh-session
$ npm install -g mistery-manager
$ mistery-manager COMMAND
running command...
$ mistery-manager (-v|--version|version)
mistery-manager/1.0.0 linux-x64 node-v12.17.0
$ mistery-manager --help [COMMAND]
USAGE
  $ mistery-manager COMMAND
...
```
